# Scripts

Esta carpeta contiene utilidades para el desarrollo, mantenimiento y depuración de Exclusive Shop AI.

## Requisitos

Ejecuta todos los scripts desde la raíz del proyecto:

python scripts/<nombre_del_script>.py

---

## crear_vectorstore.py

Genera o actualiza el índice vectorial FAISS utilizando el catálogo CSV y los documentos PDF.

python scripts/crear_vectorstore.py

---

## explorar_catalogo.py

Muestra una lista de los productos disponibles en el catálogo para verificar que fueron cargados correctamente.

python scripts/explorar_catalogo.py

---

## probar_aliases.py

Comprueba que los alias de productos se generen correctamente.

python scripts/probar_aliases.py

---

## list_models.py

Lista los modelos de inteligencia artificial disponibles para configurar el proyecto.

python scripts/list_models.py

---

## Nota

Estos scripts son herramientas de apoyo para desarrollo y mantenimiento. No forman parte de la API ni de la aplicación principal.
